# Getting Started with Safe Locker API

This guide walks you through creating an account, verifying it, setting up a password, and generating your first API key on the Safe Locker platform.

---

## Step 1: Create an Account

<Callout title="Step 1: Create Your Account" type="info">
Navigate to the Safe Locker registration page to begin the signup process.
</Callout>

**1. Fill in Your Details**:
- **First Name** and **Last Name**
- **Email Address** (required for verification)
- **Business Name**

**2. Accept Terms and Conditions**:
Check the box to accept the platform's terms and conditions.

**3. Create Account**:
Click the green **Create Account** button.

![Image of the registration form showing input fields for account creation](attachment:/mnt/data/Screenshot 2024-11-08 at 3.40.29 PM.png)

> **Tip:** Ensure your email address is correct, as you’ll need it for account verification.

---

## Step 2: Verify Your Email

<Callout title="Email Verification" type="warning">
A verification email will be sent to your provided email address.
</Callout>

**1. Check Your Inbox**:
Look for an email from Safe Locker with a verification link.

**2. Click the Verification Link**:
Clicking this link will activate your account.

**3. Resend Option**:
If you don’t receive the email, click **Resend** on the confirmation page.

![Image showing a confirmation screen with a resend option for verification email](attachment:/mnt/data/Screenshot 2024-11-08 at 3.42.59 PM.png)

> **Note:** Sometimes the email may land in your Spam or Junk folder. If you don’t see it, check there.

---

## Step 3: Set Up a Password

<Callout title="Password Setup" type="tip">
After email verification, you’ll need to create a secure password.
</Callout>

**Password Requirements**:
- Minimum 8 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one number
- At least one special character (e.g., `!@#$%^&*`)

1. **Enter a Password** that meets these criteria.
2. **Confirm Your Password** by re-entering it.
3. **Save and Sign In** to log into your account.

![Image showing the password setup screen with password criteria](attachment:/mnt/data/Screenshot 2024-11-08 at 3.47.09 PM.png)

> **Common Pitfall**: Ensure your password meets all listed requirements. The platform will highlight which criteria are missing if it doesn’t meet them.

---

## Step 4: Navigate the Dashboard

<Callout title="Dashboard Overview" type="info">
The Safe Locker Dashboard is your main interface for managing lockers, accessing reports, and generating API keys.
</Callout>

After logging in, you’ll see your **Dashboard** with options for **Locker Management**, **Reports**, **Tickets**, and more.

![Image of the main dashboard showing locker availability and usage metrics](attachment:/mnt/data/Screenshot 2024-11-08 at 3.56.56 PM.png)

> **Tip:** Familiarize yourself with the left-side menu to explore sections like **API Keys**, **Tickets**, **Reports**, and **Settings**.

---

## Step 5: Generate Your First API Key

<Callout title="Generate API Key" type="success">
You’ll need an API key to integrate with the Safe Locker API.
</Callout>

**Steps**:
1. Go to the **API Keys** section in the left menu.
2. Click on **Generate API Key**.
3. **Fill in API Key Details**:
   - **API Name**: A descriptive name for your API key (e.g., "Order Management Key").
   - **API Description**: A brief description of the API key’s purpose.
   - **Expiration Date**: Set an expiration date if needed.
4. Click **Generate** to create the API key.

![Image showing the API key generation form with fields for API name, description, and expiration](attachment:/mnt/data/Screenshot 2024-11-08 at 3.55.45 PM.png)

> **Security Notice**: Store your API key securely. It will only be displayed once. If compromised, you can regenerate a new one.

![Image showing the security notice with the generated API key](attachment:/mnt/data/Screenshot 2024-11-08 at 3.55.55 PM.png)

## Troubleshooting Common Issues

<Details summary="Click to Expand Troubleshooting Tips">
1. **Email Verification Not Received**:
   - Check your Spam folder and use the **Resend** option if needed.
2. **Password Requirements Not Met**:
   - Ensure your password matches all listed requirements.
3. **API Key Expiration**:
   - Be mindful of the expiration date set during API key creation. Generate a new key if it expires.
</Details>
